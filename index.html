Print statement debugging is a thing of the past.

Kode debug langsung dari editor. Luncurkan atau lampirkan ke aplikasi Anda yang sedang berjalan dan debug dengan titik henti sementara, tumpukan panggilan, dan konsol interaktif.

Git commands built-in.
